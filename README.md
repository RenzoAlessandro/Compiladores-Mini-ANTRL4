#### IMPLEMENTACIÓN DE UN ANALIZADOR LÉXICO + SINTÁCTICO
```
fun fibonacci(i:int):int
   if i <= 1
      return 1
   else
      return fibonacci(i - 1) + fibonacci(i - 2)
   end
end

```
